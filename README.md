## Overview

This project is a conversion-focused landing page for a fictional AI meeting assistant called Briefly. The goal was to design and build a polished marketing page with strong hierarchy, trust signals, CTA placement, responsiveness, and clean component structure.

## Features

- Responsive hero section with strong CTA
- Social proof and testimonials
- Problem/solution storytelling
- Feature and how-it-works sections
- Trust/security section
- FAQ with accordion interaction
- Final CTA section
- Metadata and favicon support

## Tech stack

- Next.js
- TypeScript
- Tailwind CSS
- Lucide React
- Vercel

## Run locally

```bash
npm install
npm run dev
```

## Design decisions

- Used a clean B2B SaaS visual direction to keep the page professional and conversion-focused.
- Chose a clear CTA hierarchy with “Start free” as the primary action.
- Structured the page as hero → proof → problem → features → trust → FAQ → final CTA to reduce friction and support conversion.
- Kept the color palette restrained and used teal for emphasis and CTA moments.
